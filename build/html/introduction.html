Introduction¶

Welcome to the Scientific Articles Analysis with Grobid documentation.

This project analyzes 10 open-access Bitcoin-related articles using the Grobid tool. It extracts structured information and presents it in different formats such as CSV and PNG.

### Main Objectives - Extract keywords and generate a word cloud from abstracts. - Analyze the number of figures per article and visualize the results. - Extract and list all hyperlinks found in the articles.

### Why Use This Project? ✅ Automates article analysis with minimal effort. ✅ Uses Grobid for high-accuracy text extraction. ✅ Provides clear visualizations and structured data outputs.
